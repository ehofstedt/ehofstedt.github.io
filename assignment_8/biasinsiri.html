Project Summary


The Problem

Siri is unreliable for users whose primary language is not English. While some have developed work-around solutions to use Siri, many are deterred by the present challenges and choose not to use it at all.


The Solution

Our solution allows Siri to learn from its users so that it may better recognize voices, accents, commands, and habits.

It also allows Siri to recover from errors with minimal user input. The user may correct Siri but they would not need to have further dialogue about the error or send in a report.
